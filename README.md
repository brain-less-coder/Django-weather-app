A simple Django app that fetches weather data using the OpenWeatherMap API.

## Features
- Search by city name
- Displays temperature, humidity, coordinates, and country
- Bootstrap styled UI

## Setup
1. Clone repo
2. Run `python manage.py runserver`
3. Visit `http://127.0.0.1:8000/`
